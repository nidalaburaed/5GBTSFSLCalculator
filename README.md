# Functional view

##### C++-based script that calculates Free Space Path Loss of a 5G New Radio

# DEMO

```bash
$ ./FSPLCalc.exe
Enter frequency (in MHz): 150
Enter distance (in km): 1.2
Free Space Path Loss (FSPL) = 77.5455 dB
```

# Instructions to compile and run the code

Compile the code using the following command:

```bash
g++ -o FSPLCalc FSPLCalc.cpp .
```

Run the code using the following command:

```bash
./FSPLCalc
```
